## 概要
HTML,CSS,JavaScript学習のアウトプットとして制作した、
架空のカフェサイトです。

## 使用技術
- HTML / CSS
- JavaScript（DOM操作）

## 実装内容
- HTML/CSSによるページ構造とスタイル設計
    セクションごとに意味のあるマークアップを行い、レスポンシブ対応をを行いました。
    メディアクエリを用いた、モバイル・タブレット・PCの画面幅に応じた
    レイアウト調整を行いました。
- JavScriptによる動的なUI制御
    アイキャッチ画像のカルーセル、
    ハンバーガーメニューの開閉、
    メニュー部分のスライダーなど、
    ユーザーの操作に応じたインタラクションを実装しました。
